# Vue Typewriter

> For Vue 1.x see the [1.x branch](https://github.com/eduardostuart/vue-typewriter/tree/1.x).

## Install

**NPM**

`npm install vue-typewriter --save`

## Usage

```vue
<typewriter
    :speed="200"
    :full-erase="true"
    :interval="300"
    :words='["Awesome!","Amazing!"]'>
    Vue Typewriter is
</typewriter>
```
