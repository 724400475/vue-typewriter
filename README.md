# Vue Typewriter

> Compatibility note: **vue-typewriter** requires **Vue 1.x**

## Install

**NPM**

`npm install vue-typewriter --save`

## Usage

```vue
<typewriter
    :speed="200"
    :full-erase="true"
    :interval="300"
    :words='["Awesome!","Amazing!"]'>
    Vue Typewriter is
</typewriter>
```
